# Lade.Fm

Lade.Fm is a Last.fm insights dashboard that turns your listening history into clear charts, trends, and shareable highlights. It includes real‑time activity views, period filters for top artists/albums/tracks, tag/genre insights, and an album collage generator.

## Features
- Real‑time charts for daily and hourly listening
- Top artists, albums, and tracks with period filters
- Genre/tag insights and diversity scoring
- Collage generator for sharing your top albums
- Mobile‑friendly responsive layout

## Getting Started
```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```

## Deployment
This project is Vercel‑ready (includes `vercel.json`). You can push to GitHub and import directly into Vercel.
